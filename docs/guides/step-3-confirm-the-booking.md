- After the slot has been reserved successfully, confirm the booking by using the Confirm a service booking API. After a successful confirmation, you will receive details about the invoice in the API response.
- After the guest has checked in, you can mark the status of an appointment as checked-in.

- If you want to undo the guest check-in, use the Undo the status of a checked-in appointment API.
