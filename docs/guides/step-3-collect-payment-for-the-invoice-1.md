### Collect cash payments in Zenoti

Updated over 3 years ago

- Verify the invoice details by using the Retrieve invoice details API.
- Use the Retrieve the saved credit/debit cards of a guest API to retrieve the list of saved cards of a guest.
- Collect payment by using the Add a credit/debit card for a guest API.
- Use the Insert custom payment for an invoice into Zenoti API to insert the details of a custom payment (which has been collected by using third-party software) for an invoice into Zenoti.
- If the guest wants to use a new card for payment, use Collect invoice payment by using a guest's unsaved credit/debit card API and complete the payment.
- If the card on file is mandatory and the guest has no saved cards, you can add it using Add a credit/debit card for a guest API.
