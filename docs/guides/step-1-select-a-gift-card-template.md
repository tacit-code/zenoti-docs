### Option 1: Predefined amount

### Option 2: Custom amount

### Option 3: Gift card with services or day packages

Try any one of the following options to select a gift card template:

Use the List all templates API to retrieve the gift card templates with predefined amounts available in the Admin > Setup > Gift Card Templates section. Choose a template and proceed to Step 2

Create a gift card template with a custom amount by using the Create a giftcard template with custom amount API and proceed to Step 2

Create a gift card template with a combination of services and packages by using the Create a gift card template with services or day packages API. (Use these two APIs to retrieve service IDs and package IDs: List all services of a center and List all packages of a center), and then proceed to Step 2.
